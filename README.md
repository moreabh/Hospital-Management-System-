Hospital Management System

>Overview

  -A web-based hospital management application built using Java, JDBC, MySQL, HTML, CSS, and JavaScript.
  -It helps manage patients, doctors, billing, and appointment scheduling efficiently.

>Technologies Used

  -Frontend: HTML, CSS, JavaScript

  -Backend: Java (JDBC)

  -Database: MySQL

  -Connector: MySQL Connector/J (JDBC Driver)
